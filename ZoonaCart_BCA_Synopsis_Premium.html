1. Introduction

ZoonaCart is a next-generation E-commerce ecosystem designed to bridge the gap between traditional online retail and modern financial flexibility. Built using a high-performance PHP/MySQL stack, the platform introduces features like a Native Digital Wallet and Dynamic HSN-based Taxation, ensuring it meets both user convenience and corporate compliance standards.

2. Objectives

User Centric Design To provide a premium "App-like" mobile interface that ensures high engagement and conversion rates.
Verified Security Implementing secure Authentication flows including OTP verification and Google OAuth integration.
Financial Automation Automating Tax calculations (GST) and managing returns through an instant Wallet refund system.
Admin Empowerment A centralized dashboard to manage inventory, sales analytics, and customer support with ease.

3. Technology Stack

HTML5 / CSS3 Core PHP 8.x MySQL RDBMS JavaScript ES6 Google Identity API SMTP Mailer HTML2Canvas

4. System Architecture

The system follows a modular 3-tier architecture, separating the Presentation (Design), Logic (PHP Engine), and Data (MySQL) layers.

System Architecture Flow

User Interface → PHP Middle-ware → MySQL Storage

The architecture is optimized for low-latency database queries and secure session management.

5. Core Modules

A. Authentication Module

Dual-verification system supporting traditional Email/OTP registration and modern Google One-Tap sign-in for reduced friction.

B. Financial Logic Module

A sophisticated module that calculates Taxes not just as a flat rate, but dynamically based on the HSN Code of individual products, reflecting real-world business scenarios.

C. Wallet & Transaction Engine

A ledger-based wallet system that tracks every 'Debit' and 'Credit'. This module handles instant refunds into the wallet upon return approval, a feature commonly found in Tier-1 E-commerce apps.

6. Project Innovation

Unlike standard CRUD applications, ZoonaCart incorporates several innovations:

  • Amazon-Style Invoices: High-fidelity PDF generation with professional invoice bifurcations.
  • Responsive Glassmorphism: A UI that adapts fluidly between 4K desktops and compact smartphones.
  • Auto-Healing Database: Scripts that automatically patch table structures to prevent runtime errors.

7. Conclusion

ZoonaCart serves as a comprehensive demonstration of full-stack engineering. By integrating complex business logic like dynamic taxing and wallet-ledgers with a premium frontend design, the project proves that small-scale applications can achieve enterprise-level quality and reliability.
